# Copa 2026 — Jogos do Brasil

Página estática e responsiva em português para apresentar uma agenda demonstrativa da Seleção Brasileira na Copa 2026.

## Recursos

- cartões dos próximos jogos;
- tabela demonstrativa do grupo;
- visual inspirado nas cores do Brasil;
- layout adaptado para celular e computador;
- metadados Open Graph para compartilhamento.

> As datas, horários, estádios e grupos exibidos são demonstrativos. Consulte os canais oficiais antes de publicar informações de competição.

## Desenvolvimento

Requer Node.js 22 ou superior e pnpm.

```bash
pnpm install
pnpm dev
```

Para validar a versão de produção:

```bash
pnpm build
```
